Template Driven Form

Here we will create a basic login form

Above we have created a basic form but on clicking of submmit button there is no action which is to be performed, so now we will add a funciton which will be called on clicking of our submit button. Now we will bind form's "ngSubmit"" event property to the our form component's function (here we are binding with onSubmit() funciton which is writtern in our component.ts file). In form tag write : (ngSubmit)="onSubmit()". There is alot you can do with your form. till now we are done with the basic. you can use ngModel attribute to create data bindings for reading and writing input-control values. it can be used in 3 ways : 1) simple ngModel attribute 2) [ngModel] : model to template binding 3) (ngModel) : template to model binding For 2 way binding, you can write [(ngModel)]="". this will help you in both reading and writing.
